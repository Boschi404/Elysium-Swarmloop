# T01 Mathematical Reasoning — Linear Equations System

## System

```
 2x + 3y -  z =  8    (1)
 -x + 4y + 2z =  5    (2)
 3x -  y + 4z = 12    (3)
```

## Step 1 — Matrix Form

Write the system as **A · X = b**:

```
A = |  2   3  -1 |      X = | x |      b = |  8 |
    | -1   4   2 |          | y |          |  5 |
    |  3  -1   4 |          | z |          | 12 |
```

## Step 2 — Compute Determinant D = det(A)

Using Sarrus' rule:

```
D = 2·4·4 + 3·2·3 + (-1)·(-1)·(-1) - (-1)·4·3 - 2·2·(-1) - 3·(-1)·4

  = 32  +  18  +  (-1)    -  (-12)  -  (-4)   -  (-12)

  = 32 + 18 - 1 + 12 + 4 + 12

  = 77
```

**D = 77**  (≠ 0 → unique solution)

## Step 3 — Compute Dx, Dy, Dz (Cramer's Rule)

### Dx — replace column 1 with b

```
A_x = |  8   3  -1 |
      |  5   4   2 |
      | 12  -1   4 |

Dx = 8·4·4 + 3·2·12 + (-1)·5·(-1) - (-1)·4·12 - 8·2·(-1) - 3·5·4

   = 128  +  72  +  5       -  (-48)  -  (-16)  -  60

   = 128 + 72 + 5 + 48 + 16 - 60

   = 209
```

### Dy — replace column 2 with b

```
A_y = |  2   8  -1 |
      | -1   5   2 |
      |  3  12   4 |

Dy = 2·5·4 + 8·2·3 + (-1)·(-1)·12 - (-1)·5·3 - 2·2·12 - 8·(-1)·4

   = 40  +  48  +  12      -  (-15)  -  48  -  (-32)

   = 40 + 48 + 12 + 15 - 48 + 32

   = 99
```

### Dz — replace column 3 with b

```
A_z = |  2   3   8 |
      | -1   4   5 |
      |  3  -1  12 |

Dz = 2·4·12 + 3·5·3 + 8·(-1)·(-1) - 8·4·3 - 2·5·(-1) - 3·(-1)·12

   = 96  +  45  +  8       -  96  -  (-10)  -  (-36)

   = 96 + 45 + 8 - 96 + 10 + 36

   = 99
```

## Step 4 — Solve for x, y, z

```
x = Dx / D = 209 / 77 = 19/7 ≈ 2.714286
y = Dy / D =  99 / 77 =  9/7 ≈ 1.285714
z = Dz / D =  99 / 77 =  9/7 ≈ 1.285714
```

## Step 5 — Verification

| Equation | Substitution | Result | Expected |
|----------|-------------|--------|----------|
| (1) 2x + 3y − z | 2·19/7 + 3·9/7 − 9/7 = (38+27−9)/7 = 56/7 | **8** | 8 ✅ |
| (2) −x + 4y + 2z | −19/7 + 4·9/7 + 2·9/7 = (−19+36+18)/7 = 35/7 | **5** | 5 ✅ |
| (3) 3x − y + 4z | 3·19/7 − 9/7 + 4·9/7 = (57−9+36)/7 = 84/7 | **12** | 12 ✅ |

## Final Answer

```
x = 19/7  ≈ 2.714286
y =  9/7  ≈ 1.285714
z =  9/7  ≈ 1.285714
```
